## 🎯 Design Decisions Made

### 1. Time Period: 2010-2024 ✅
- **Why**: Golden period starts 2010, includes recent 2022-2024 trends
- **Trade-off**: 2024 data incomplete but captures post-COVID recovery
- **Implementation**: Flexible filtering in notebooks

### 2. Outlier Treatment: Flag + Transform ✅  
- **Why**: Trust World Bank API accuracy, outliers may be real phenomena
- **Approach**: No deletion, separate flags for analyst review
- **Benefit**: Preserves all data, enables robust analysis

### 3. Missing Values: KNN Imputation ✅
- **Why**: Better than mean, respects country similarity (k=5)
- **Fallback**: Forward/backward fill if KNN insufficient
- **Result**: 100% complete dataset for ML

### 4. Scaling: StandardScaler ✅
- **Problem**: GDP (0-174k) vs Internet (0-100) incompatible scales
- **Solution**: (x - mean) / std normalization
- **Impact**: ML algorithms now use all features equally

### 5. Clustering: K-Means (k=6) ✅
- **Method**: Elbow curve optimization
- **Result**: Development-stage-based clusters
- **Output**: Vietnam assigned to meaningful peer group
