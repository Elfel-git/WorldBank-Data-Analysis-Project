## 📊 Pipeline Architecture

### Phase Overview

```
┌─────────────────────────────────────────────┐
│ PHASE 0: DATA INTEGRATION                   │
│ Input: ../nam/*.csv                         │
│ Output: dataset_merged.csv                  │
└────────────────┬────────────────────────────┘
                 │
                 ▼
┌─────────────────────────────────────────────┐
│ PHASE 1: AUTO PROFILING & DIAGNOSTICS      │
│ Input: dataset_merged.csv                  │
│ Output: diagnostic reports + visualizations │
│         draft_config.yaml (auto-generated)  │
└────────────────┬────────────────────────────┘
                 │
                 ▼
┌─────────────────────────────────────────────┐
│ PHASE 1.5: HUMAN REVIEW & CONFIG (MANUAL)  │
│ Action: Review reports, edit config        │
│ Output: final_config.yaml (edited)         │
└────────────────┬────────────────────────────┘
                 │
                 ▼
┌─────────────────────────────────────────────┐
│ PHASE 2: EXECUTION PIPELINE                 │
│ Input: dataset_merged.csv + final_config    │
│ Processing:                                 │
│   1. Log Transform                          │
│   2. Outlier Handling                       │
│   3. Scaling & Normalization                │
│   4. KNN Imputation                         │
│ Output: dataset_final.csv + scaler_model.pkl│
└─────────────────────────────────────────────┘
```

### Module Dependencies
- **config_handler:** No dependencies (utilities only)
- **data_integration:** pandas, numpy
- **diagnostics:** pandas, numpy, matplotlib, seaborn, json, logging
- **processing:** pandas, numpy, scikit-learn, pickle, logging

## 🚀 Getting Started

### Quick Start (5 minutes)

1. **Open PHASE 0 notebook:**
   ```
   File: 00_data_integration.ipynb
   Action: Run all cells (Shift+Enter)
   Output: Check outputs/dataset_merged.csv
   ```

2. **Open PHASE 1 notebook:**
   ```
   File: 01_auto_profiling.ipynb
   Action: Run all cells
   Output: Check diagnostic reports and visualizations
   ```

3. **Review and configure:**
   ```
   File: outputs/draft_config.yaml
   Action: Copy to outputs/final_config.yaml and edit
   Decision: Enable/disable transformations based on diagnostics
   ```

4. **Open PHASE 2 notebook:**
   ```
   File: 02_execution_pipeline.ipynb
   Action: Run all cells
   Output: Check outputs/dataset_final.csv (FINAL PRODUCT!)
   ```

### Requirements
- Python 3.8+
- pandas, numpy, scikit-learn, matplotlib, seaborn, pyyaml
- Jupyter Notebook or JupyterLab

### Installation
```bash
pip install pandas numpy scikit-learn matplotlib seaborn pyyaml jupyter
```

## 🔧 Configuration Strategies

### Recommended Configurations

**Strategy 1: Conservative (minimal changes)**
```yaml
log_transform: {enabled: false}
outlier_detection: {iqr_multiplier: 2.0, action: clip}
scaling: {method: standard}
imputation: {n_neighbors: 5}
```

**Strategy 2: Moderate (balanced - RECOMMENDED)**
```yaml
log_transform: {enabled: true}
outlier_detection: {iqr_multiplier: 1.5, action: clip}
scaling: {method: standard}
imputation: {n_neighbors: 5}
```

**Strategy 3: Aggressive (heavy transformations)**
```yaml
log_transform: {enabled: true}
outlier_detection: {method: isolation_forest, action: impute}
scaling: {method: robust}
imputation: {n_neighbors: 3}
```
